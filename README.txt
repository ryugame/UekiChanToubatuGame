			◆ゲーム概要
・操作方法
	WASD、または矢印キーで蘭子が動きます。
	Fキーで炎、Bキー長押しでビームが出ます。
	ゲーム中にEscキー長押しでタイトル画面に戻ることができます
・遊び方
	ゲーム内にて書いております、ご確認ください。
・画面改造度について
	フルスクリーンで表示するとおもすぎてプレイできないので解像度を小と中に分けました、
	タイトル画面左下の設定タブから調節できます。
・難易度について
	同じくタイトル画面左下の設定タブより難易度をEasy、Normal、Hardよりお選びできます。
	詳しく説明すると

			Easy		Normal	   Hard	

プレイヤーHP		200		150	   120
敵HP			1000		2500	   5000	
ぴにゃこらたの数		１		１	   1

になります
